1. E-Commerce-Sales-Analysis
I built an end-to-end SQL-based E-Commerce Sales Analysis project where I designed the schema, loaded data, and used advanced SQL queries to extract business insights.



2. Overview
This project analyzes e-commerce sales data using SQL to answer real business questions related to revenue, customers, and products.

3. Tools Used
- MySQL

4. Skills Demonstrated
- Joins
- GROUP BY & HAVING
- Subqueries
- Window Functions (RANK)
- Business Analysis

5. Key Analysis
- Total & monthly revenue
- Top customers & products
- Category-wise sales
- Customer ranking
- Repeat customer analysis

6. How to Run
1. Run `schema.sql`
2. Run `data.sql`
3. Run `analysis_queries.sql`
